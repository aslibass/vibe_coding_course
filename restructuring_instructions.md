### Step 2: Generate the `course.json` Manifest
- Create a `course.json` file in the root directory. This acts as the source of truth for the headless React front-end.

### Step 3: Native Markdown Structuring (Premium Blocks)
The ZeroPoint engine automatically converts standard markdown into premium UI components. You MUST format the markdown using these rules to trigger the correct UI:

1. **Hero Sections**
   - The engine automatically extracts the very first `# H1 Heading` and the immediately following text paragraph to use as the full-width Hero Banner for the page.

2. **Mission/Alignment Blocks**
   - Any standard markdown blockquote (`> Your text here`) will be automatically transformed into a giant, high-impact glassmorphism alignment card. Use these strictly for core mission statements, key principles, or profound quotes.
3. **Video Players**
   - Any standard markdown link to YouTube (e.g., `[Watch Lesson](https://youtube.com/...)`) will be intercepted and rendered as a premium embedded video player inline.
4. **Code Blocks**
   - Ensure all code snippets use standard triple backticks with the correct language tag (e.g., ` ```typescript `) to trigger syntax highlighting and copy-to-clipboard functionality.
5. **Table of Contents (Right Rail)**
   - The engine automatically generates a sticky, dynamic Table of Contents mapping based on your headings. 
   - You MUST structure your documents hierarchically using `##` (Section) and `###` (Sub-section) markdown headings for the ScrollSpy TOC to map the user's reading position accurately.
