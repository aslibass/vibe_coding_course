# ✨ Vibe Coding: Zero to Point Solutions

Welcome to the **Vibe Coding Course Repository**. 

This syllabus was explicitly designed for ministry leaders, volunteers, and church groups with *absolutely zero coding background* who want to use AI to build beautiful, mission-aligned software to serve their communities.

We do not teach you the syntax of coding. We teach you **Precision of Intent** and the **Iteration Loop**, letting AI tools do the heavy technical lifting.

---

## 📅 The Core Curriculum (4 Weeks)

This course is designed to be taken over four separate, 2-hour group sessions.

*   📂 **[00_PRE_FLIGHT](./00_PRE_FLIGHT)**: Your Week 0 preparation. Setting up Google AI Studio, GitHub, and Node.js on your machine.
*   📂 **[01_SESSION_IDENTITY](./01_SESSION_IDENTITY)**: Learning the "Director Mindset" and building your first Ministry Bio Site using Master Prompts.
*   📂 **[02_SESSION_ITERATION](./02_SESSION_ITERATION)**: Learning the T-A-O (Thought-Action-Observation) Loop, screenshot debugging, and fixing AI hallucinations.
*   📂 **[03_SESSION_LOGIC](./03_SESSION_LOGIC)**: Breaking out of read-only apps. Designing a "Praise Report" system and hooking it up to a real Google Firebase database.
*   📂 **[04_SESSION_GRADUATION](./04_SESSION_GRADUATION)**: Locking down your database with Authentication and deploying your finished project live to the internet.

---

## 🚀 The Post-Graduation Tracks (Level Up)

Once you understand the basics of Vibe Coding (copy/pasting code from AI Studio to your local machine), you can unlock **Agentic Workflows**. These modules are for advanced graduation students:

*   📂 **[05_LEVEL_2_ANTIGRAVITY](./05_LEVEL_2_ANTIGRAVITY)**: Shifting from a "Director" (using AI Studio) to a "Manager" overseeing **Antigravity** (an autonomous Agent that edits files for you).
*   📂 **[06_LEVEL_3_GITHUB](./06_LEVEL_3_GITHUB)**: Mastering version control, learning to save "snapshots," and collaborating with your team safely.
*   📂 **[07_LEVEL_4_RAILWAY](./07_LEVEL_4_RAILWAY)**: Deploying enterprise-level, full-stack architectures using Railway.
