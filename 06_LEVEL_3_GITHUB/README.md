## 🕰️ Core Concept: The Time Machine

In Week 0, we created a GitHub account and treated it like a backup drive. But GitHub is far more powerful. It is a **Version Control System**.

Imagine writing a book. Instead of saving files like `final.doc`, `final_final.doc`, and `really_final_this_time.doc`, Version Control takes a pure snapshot of your entire project at exact moments in time.

If Antigravity accidentally deletes your beautiful homepage CSS, you do not have to panic or try to prompt it back into existence. You simply tell GitHub: *"Put my code back exactly the way it was yesterday at 4:00 PM."*

By mastering GitHub, you remove all fear from coding. You can experiment wildly, knowing you can always undo any mistake with a single command.
